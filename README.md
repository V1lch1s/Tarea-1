# Descripción ¿qué hice?
Estuve estudiando la sintaxis de C y técnicas para simular la Programación Orientada a Objetos de C++ en C con structs, punteros a funciones y estructras opacas.

Estuve revisando los diferentes conceptos al mismo tiempo que desarrollaba lo que me pedían y practicaba el entendimiento de código de mediano nivel. Para esto conté con la ayuda de diversas platformas de Inteligencia Artificial disponibles en internet, y algo de validación con otras fuentes (Libros, videos de YouTube). Al final el tiempo se me fue y trabajé de más como parte de la experiencia de aprendizaje.
